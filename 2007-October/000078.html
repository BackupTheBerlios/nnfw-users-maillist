Ciao Eduardo,
yes... now you don't need to allocate the OutputFunction on the Heap...
this means that you should use this notation:

inputLayer->setFunction( SigmoidFunction(1.0) );

Note that there is no "new"... this means a better use of memory... in
fact now you can use a local variable and pass it to more that on layer:

SigmoidFunction s(1.0);
layer1->setFunction( s );
layer2->setFunction( s );
...
layern->setFunction( s );
