Quickly, I can answer to your second question because I've  
experienced the same problem few months ago.
You've simply to use the DotLinker class (http://www.nnfw.org/api/ 
classnnfw_1_1DotLinker.html) instead of the old MatrixLinker.

eg (a snap-shot from my simulation):

// Synaptic connections (input->hidden)
nnfw::DotLinker *TargetDistanceToHidden, *TargetAngleToHidden,  
*USPerceptionToHidden;
// Synaptic connections (hidden->output)
nnfw::DotLinker *HiddenToOutput_Steering, *HiddenToOutput_Detonation;

// Set up the matrix linkers
TargetDistanceToHidden = new nnfw::DotLinker 
(inputLayer_TargetDistance, hiddenLayer);
TargetAngleToHidden = new nnfw::DotLinker(inputLayer_TargetAngle,  
hiddenLayer);
USPerceptionToHidden = new nnfw::DotLinker(inputLayer_USPerception,  
hiddenLayer);
HiddenToOutput_Steering = new nnfw::DotLinker(hiddenLayer,  
outputLayer_Steering);
HiddenToOutput_Detonation = new nnfw::DotLinker(hiddenLayer,  
outputLayer_Detonation);
